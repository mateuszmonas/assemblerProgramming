# assemblerProgramming
projects made for assembler languages class

## [project 1](./szyfr.asm)
Program to encrypt files with XOR function

usage:

```bash
szyfr.exe file_in file_out "key"
```

## [project 2](./zoom.asm)
Program to zoom in on text using VGA mode

usage:

```bash
zoom.exe zoom text
```
